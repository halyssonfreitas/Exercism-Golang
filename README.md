# Exercism Golang

This is a repository to store my progress in Exercism platform.

See [Exercism Golang](https://exercism.org/tracks/go)

## About Exercism way

Each exercism can be done online and locally. This repository is a result when execute locally.

See instruction in [CLI](https://exercism.org/settings/api_cli)
